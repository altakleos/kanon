# Plan: Bootstrap `agent-sdd` v0.1

## Context

This plan records the execution of the v0.1 bootstrap — the first release of the portable, self-hosting SDD kit. The full design memo (including agent debate, counter-arguments, and resolved scope) lives in the session plan at `~/.claude/plans/implement-1-thru-7-whimsical-backus.md`; this file is the in-repo execution record.

The bootstrap resolves a chicken-and-egg: the kit requires itself to develop itself. The resolution is a named pre-SDD bootstrap phase (commits 1–3), documented in ADR-0002, after which SDD engages for all subsequent work.

## Acceptance Criteria

- [ ] AC1: All self-hosting assertions from the session plan hold (see `~/.claude/plans/...`):

  2. `agent-sdd init <target> --tier <N>` + `verify` OK for all four tiers

  4. Tier subset invariant (tier-N ⊆ tier-(N+1))

  6. Tier migration round-trip preserves user content
- [ ] AC2: CI green — ruff, mypy --strict, pytest (coverage ≥80%), check_foundations, check_links, check_package_contents, check_template_consistency
- [ ] AC3: `v0.1.0a1` tag pushed to origin
